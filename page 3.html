html heading

  • HTML Headings Are Titles Or Subtitles That You Want To Display On A Webpage.
  • HTML Headings Are Defined With The < H1> To < H6> Tags.
  • < H1> Defines The Most Important Heading. < H6> Defines The Least Important Heading.
  • < h1> headings should be used for main headings, followed by < h2> headings, then the less important , and so on.

html formating

  • < b> - Bold text
  • < strong> - Important text
  • < i> - Italic text
  • < em> - Emphasized text
  • < mark> - Marked text
  • < small> - Smaller text
  • < del> - Deleted text
  • < ins> - Inserted text
  • < sub> - Subscript text
  • < sup> - Superscript text
  • < u> - underline text
